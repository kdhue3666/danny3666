# Sudoku

Official website files for the Sudoku mobile app.

Sudoku is a calm mobile puzzle game with easy, normal, and hard modes, daily challenges, notes, hints, statistics, achievements, and a focused mobile-first game experience.

## Pages

- `index.html` - App website and support page
- `privacy-policy.html` - Privacy Policy for Google Play and users
- `app-ads.txt` - Authorized seller file for Google AdMob
